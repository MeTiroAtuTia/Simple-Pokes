[size=13pt][color=red]Introduction[/color][/size]
[size=8pt]Simple Pokes provides a very simple functionality that emulates the FB Pokes system, and takes advantage of the SMF 2.1 Alerts system for a better interaction with the mod.[/size]

[hr]

[size=13pt][color=red]Features[/color][/size]
[list]
[li][size=11pt]Send pokes[/size][/li]
[li][size=11pt]Ignore pokes[/size][/li]
[li][size=11pt]Disable/enable alerts when receiving a poke[/size][/li]
[/list]
